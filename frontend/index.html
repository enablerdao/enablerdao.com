(Mission)

極上を、おすそわけ。

熱海・北海道・ハワイにある5つの別荘に、誰でも泊まれます。
Airbnbで予約するだけ。オーナーが惚れ込んだ特別な場所を、あなたとシェアします。

RWA（実物資産）× OSS × トークンエコノミー。
貢献にインセンティブを設計した分散型コミュニティプロトコル。

(Philosophy)

enablerが大事にしていること

enabler.config.ts — Core Protocol Values

enabler.values.js 
const enabler = {

  share:          "いいもの見つけたら、みんなで楽しむ",
  // 別荘も体験も、分け合うほど豊かになる

  respect:        "あなたの最高は、あなたが決める",
  // 押し付けない。でもいい場所は教える

  experience:     "まず泊まってみる。話はそれから",
  // 百聞は一見に如かず

  trust:          "言い出しっぺが、最後までやる",
  // だから安心して任せられる

};

export default enabler;
enabler.config.ts 
const PROTOCOL_VALUES = {

  openSource:     "fork it, improve it, ship it",
  // all repos are public. PRs earn EBR tokens

  decentralized: "no single point of failure",
  // governance by token holders, not a CEO

  rwa:            "real-world assets, on-chain access",
  // 5 properties tokenized as member perks

  permissionless: "contribute first, ask later",
  // ship code → earn tokens → unlock perks

} as const;

export default PROTOCOL_VALUES;

(Story)

        
バグを直す。波を待つ。
どちらも、世界を美しくする作業だ。

        
「場所を選ぶ」から
「場所を創る」へ。

        

          
いい風が吹く場所を見つけたら、

          そこに最高の空間をつくる。

          そして、みんなで使う。

          それだけ。シンプル。

        

      

    



    

      

        
(Message)

        

          
「少しだけ、
立ち止まってみませんか。」

          
僕が惚れ込み、自分でも手に入れた特別な場所。

          そこは日常のノイズから離れ、ただ静かに自分を取り戻せる空間です。

          
なぜ、ここをあなたにシェアしたいのか。

          それは、美しい建築やソフトウェアのように、時代を超えて残る「本当にいいもの」を世の中に増やしていきたいからです。

          次の世代へ繋がる、より豊かなライフスタイルの形を。

          
そんな未来へ続く時間をみんなで創っていくために。

          まずは一度、この場所で過ごす心地よさを体験してみませんか？
